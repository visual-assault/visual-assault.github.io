# http://visual-assault.org/

Website for visual assault, llc.

## Requirements

* Ruby 2.0.0+
* Bundle `gem install bundler`

## Get Started
Site is built with [Jekyll](http://jekyllrb.com) and served with GitHub Pages.

``` bash
# To install dependencies
./bin/install.sh
# Run Jekyll
./bin/run.sh
```
